`FilterBuilder` is the editor for one filter condition. Given the available
`fields`, it renders a field picker, an operator picker scoped to that field's
type (e.g. "contains", "is any of", "between"), and the right value input —
text, number, tag list, date range, or faceted select — automatically. It is
controlled: it calls `onChange` with a partial patch as the user edits and
`onRemove` to delete the condition. `FilterBar` composes this internally; use it
directly to build a custom filter row.

## Example

<Demo>
  <FilterBuilderDemo />
</Demo>

```tsx
import { FilterBuilder } from '@officina/ui';

<FilterBuilder
  condition={condition}
  fields={[
    { id: 'name', label: 'Name', type: 'string' },
    { id: 'createdAt', label: 'Created', type: 'date' },
  ]}
  onChange={(id, patch) => updateCondition(id, patch)}
  onRemove={(id) => removeCondition(id)}
/>;
```
