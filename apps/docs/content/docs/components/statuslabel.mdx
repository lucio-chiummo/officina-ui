`StatusLabel` pairs a tone-coded dot with a text status.

<Demo>

</Demo>

```tsx
import { StatusLabel } from '@officina/ui';

<StatusLabel tone="success" pulse>Online</StatusLabel>
<StatusLabel tone="danger" variant="soft">Offline</StatusLabel>
```
