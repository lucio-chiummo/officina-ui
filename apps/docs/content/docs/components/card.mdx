`Card` is the base surface for grouped content — a rounded, bordered box with a
subtle shadow and an adjustable `padding` scale (`none`, `sm`, `md`, `lg`). Drop
any content inside, or compose it with `CardHeader`, `CardTitle`,
`CardDescription`, `CardContent`, and `CardFooter` for a structured layout.

## Example

<Demo>
  <CardDemo />
</Demo>

```tsx
import { Card, CardHeader, CardTitle, CardContent } from '@officina/ui';

<Card>
  <CardHeader>
    <CardTitle>Revenue</CardTitle>
  </CardHeader>
  <CardContent>$24.8k this month</CardContent>
</Card>;
```
