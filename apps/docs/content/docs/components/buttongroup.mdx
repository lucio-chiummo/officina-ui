---
title: ButtonGroup
description: Clusters related buttons into one attached horizontal or vertical unit with seamless edges.
---

import { ButtonGroupDemo } from '@/components/demos/buttons-demo';

`ButtonGroup` lays out a set of related `Button`s as a single connected unit. By
default the children are `attached` — touching corners are squared off so the
group reads as one control — and `orientation` switches between a horizontal row
and a vertical stack. It is a presentational wrapper (`role="group"`); each
button keeps its own handlers.

## Example

<Demo>
  <ButtonGroupDemo />
</Demo>

```tsx
import { ButtonGroup, Button } from '@officina/ui';

<ButtonGroup>
  <Button variant="secondary">Day</Button>
  <Button variant="secondary">Week</Button>
  <Button variant="secondary">Month</Button>
</ButtonGroup>;
```
