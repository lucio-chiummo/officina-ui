`Scheduler` is a full calendar with day, week, month, and agenda views. Users can
drag events to move them and drag their edges to resize, while clicking an empty
slot fires `onSlotSelect` for quick creation. State is controlled: pass `events`
and an `onEventsChange` handler that persists edits. Set `defaultView`, restrict
the available `views`, supply `resources` for lane-based scheduling, or set
`readOnly` to disable editing.

## Example

<Demo>
  <SchedulerDemo />
</Demo>

```tsx
import { Scheduler } from '@officina/ui';
import { useState } from 'react';

function Calendar() {
  const [events, setEvents] = useState([
    { id: 'evt-1', title: 'Design review', start: new Date(), end: new Date() },
  ]);
  return (
    <Scheduler
      events={events}
      onEventsChange={setEvents}
      defaultView="week"
      onSlotSelect={({ start, end }) =>
        setEvents((prev) => [...prev, { id: `evt-${Date.now()}`, title: 'New event', start, end }])
      }
    />
  );
}
```
