title: SearchFilter

import { SearchFilterDemo } from '@/components/demos/composed-demo';

`SearchFilter` is a thin wrapper around `SearchInput` purpose-built for
filter bars: it reports every keystroke through `onChange` (so the input
stays responsive) while also exposing a separate `onDebouncedChange` you can
wire to an actual query or API call, avoiding a request per keystroke.

## Example

<Demo>
  <SearchFilterDemo />
</Demo>

```tsx
import { SearchFilter } from '@officina/ui';
import { useState } from 'react';

function CustomersFilterBar() {
  const [value, setValue] = useState('');
  return (
    <SearchFilter
      value={value}
      onChange={setValue}
      onDebouncedChange={(query) => console.log('fetch with', query)}
      debounceMs={300}
      placeholder="Search customers"
    />
  );
}
```
