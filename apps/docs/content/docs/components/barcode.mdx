import { BarcodeDemo } from '@/components/demos/misc-demo';

`Barcode` turns a string into a vertical-bar SVG barcode — a decorative,
scannable-style representation for SKUs, tickets, and order ids. Pass the `value`
to encode and set `height` for the bar height in pixels (defaults to `72`). Note
it renders a deterministic bit pattern of the input rather than a specific retail
symbology like Code 128.

## Example

<Demo>
  <BarcodeDemo />
</Demo>

```tsx
import { Barcode } from '@officina/ui';

<Barcode value="SKU-48823-A" height={72} />;
```
