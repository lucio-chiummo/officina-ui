title: ListItemButton
description: Interactive, focusable row button with a selected state for menus and nav.
---

import { ListItemButtonDemo } from '@/components/demos/composed-demo';

`ListItemButton` makes a `ListItem` clickable: it renders a full-width `<button>`
with hover and keyboard focus styling and a `selected` prop that applies the
active/accent treatment for the current route or chosen option. Use it inside a
`ListItem` for navigation menus, option pickers, and any list where rows respond
to clicks. It forwards all native button props, so wire up `onClick` directly.

## Example

<Demo>
  <ListItemButtonDemo />
</Demo>

```tsx
import { List, ListItem, ListItemButton } from '@officina/ui';

<List>
  <ListItem>
    <ListItemButton selected onClick={() => navigate('/dashboard')}>
      Dashboard
    </ListItemButton>
  </ListItem>
  <ListItem>
    <ListItemButton onClick={() => navigate('/settings')}>Settings</ListItemButton>
  </ListItem>
</List>;
```
