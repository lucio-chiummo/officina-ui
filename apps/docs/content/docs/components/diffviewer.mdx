---
title: DiffViewer
description: Side-by-side before/after text diff with added and removed lines highlighted.
---

import { DiffViewerDemo } from '@/components/demos/data-display-demo';

`DiffViewer` compares `oldValue` and `newValue` line by line and renders them in
two columns — "Before" and "After" — highlighting removed lines in red and added
lines in green. Use it to show config changes, edit history, or revision
previews.

## Example

<Demo>
  <DiffViewerDemo />
</Demo>

```tsx
import { DiffViewer } from '@officina/ui';

function ConfigDiff() {
  return <DiffViewer oldValue={'status: draft\nprice: 99'} newValue={'status: live\nprice: 129'} />;
}
```
