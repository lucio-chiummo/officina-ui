---
title: FormHelperText
description: Helper/description text for a FormControl, linked via aria-describedby with tone options.
---

import { FormHelperTextDemo } from '@/components/demos/composed-demo';

`FormHelperText` renders the small description under a control. Inside a
`FormControl` it adopts the control's description id so it is linked through
`aria-describedby`. Use the `tone` prop (`muted`, `success`, or `danger`) to
match the message to the field's state.

## Example

<Demo>
  <FormHelperTextDemo />
</Demo>

```tsx
import { FormControl, FormLabel, FormHelperText, Input } from '@officina/ui';

<FormControl>
  <FormLabel>API key</FormLabel>
  <Input />
  <FormHelperText tone="muted">Find this in your account settings.</FormHelperText>
</FormControl>;
```
