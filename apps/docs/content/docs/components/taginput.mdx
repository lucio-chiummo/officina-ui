---
title: TagInput
description: Token input for entering a list of tags with delimiters, validation, and suggestions.
---

import { TagInputDemo } from '@/components/demos/inputs-demo';

`TagInput` collects a list of free-form tags. It is controlled — `value` is a
`string[]` and `onChange` fires on every add or remove. A tag commits on `Enter`
or whenever the typed text matches the `delimiter` regex (comma, semicolon,
newline, or tab by default), `Backspace` on an empty field removes the last tag,
and duplicates are ignored. Cap the count with `maxTags`, reject entries with a
`validate` callback, and offer `suggestions` that surface while typing.

## Example

<Demo>
  <TagInputDemo />
</Demo>

```tsx
import { TagInput } from '@officina/ui';
import { useState } from 'react';

function Example() {
  const [tags, setTags] = useState<string[]>(['design']);
  return (
    <TagInput
      value={tags}
      onChange={setTags}
      placeholder="Add a tag…"
      suggestions={['design', 'engineering', 'product']}
    />
  );
}
```
