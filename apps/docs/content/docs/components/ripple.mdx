title: Ripple
description: Wraps a pressable surface with a material-style click ripple from the pointer.
---

import { RippleDemo } from '@/components/demos/misc-demo';

`Ripple` wraps interactive content and emits a material-style ripple from the
exact point where the pointer presses down. Drop it around buttons, list rows,
or any pressable surface for tactile feedback. Customize the ripple `color` or
turn it off per-instance with `disabled`. The effect is purely decorative and
skipped under `prefers-reduced-motion`.

## Example

<Demo>
  <RippleDemo />
</Demo>

```tsx
import { Ripple } from '@officina/ui';

<Ripple className="rounded-md">
  <button className="px-4 py-2">Click me</button>
</Ripple>;
```
