---
title: Tooltip
description: Hover- and focus-triggered label that floats beside an element, with delay and arrow.
---

import { TooltipDemo } from '@/components/demos/overlays-demo';

`Tooltip` shows a short, non-interactive label when the user hovers or focuses
its child element. It opens after a short `delayMs`, portals the label with an
arrow pointing at the trigger, sets `role="tooltip"`, and closes on mouse leave,
blur, or `Escape`. Wrap exactly one focusable element and pass the label via
`content`. Use it for icon-button hints and terse explanations — not for
clickable content.

## Example

<Demo>
  <TooltipDemo />
</Demo>

```tsx
import { Tooltip } from '@officina/ui';

<Tooltip content="Archive" side="top">
  <button aria-label="Archive">📁</button>
</Tooltip>;
```
