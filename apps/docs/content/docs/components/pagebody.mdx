title: PageBody

import { PageBodyDemo } from '@/components/demos/composed-demo';

`PageBody` is the main content region of a page layout. It stacks its children
vertically with consistent spacing and applies a subtle fade-in on mount. Place
it below a [`PageHeader`](/docs/components/pageheader) and fill it with
[`PageSection`](/docs/components/pagesection) blocks to assemble a standard
page.

## Example

<Demo>
  <PageBodyDemo />
</Demo>

```tsx
import { PageHeader, PageBody, PageSection } from '@officina/ui';

<>
  <PageHeader title="Settings" />
  <PageBody>
    <PageSection title="Profile">{/* … */}</PageSection>
    <PageSection title="Billing">{/* … */}</PageSection>
  </PageBody>
</>;
```
