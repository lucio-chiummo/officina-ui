---
title: InputGroup
description: Wraps an input with leading/trailing addons like icons, units, or fixed prefixes.
---

import { InputGroupDemo } from '@/components/demos/inputs-demo';

`InputGroup` attaches a `prefix` and/or `suffix` addon to a control, joining them
into a single bordered shape that shows a focus ring when the inner input is
focused. Use it for fixed prefixes like `https://`, currency or unit labels, or
inline icon buttons. Place an `Input` (or any input) as the child; the group
strips the child's own border so it blends in.

## Example

<Demo>
  <InputGroupDemo />
</Demo>

```tsx
import { InputGroup, Input } from '@officina/ui';

<InputGroup prefix="https://" suffix=".com">
  <Input placeholder="your-site" />
</InputGroup>;
```
