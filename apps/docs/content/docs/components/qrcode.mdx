---
title: QRCode
description: Renders a string as a scannable QR code SVG with configurable size and accessible label.
---

import { QRCodeDemo } from '@/components/demos/misc-demo';

`QRCode` encodes any string — a URL, token, or contact payload — into a crisp,
scannable SVG QR code framed in a bordered tile. Pass the `value` to encode, set
`size` for the module dimension in pixels, and give a `label` describing the code
for screen readers. It forwards any other `div` props and merges `className`.

## Example

<Demo>
  <QRCodeDemo />
</Demo>

```tsx
import { QRCode } from '@officina/ui';

<QRCode value="https://officina.dev" size={160} label="Link to officina.dev" />;
```
