`CardTitle` is the card's heading — a semantic `<h3>` with the right size and
weight for a card surface. Put it inside a `CardHeader`, typically above a
`CardDescription`. Pass the heading text as children.

## Example

<Demo>
  <CardTitleDemo />
</Demo>

```tsx
import { Card, CardHeader, CardTitle, CardDescription } from '@officina/ui';

<Card>
  <CardHeader>
    <div>
      <CardTitle>Acme Operations</CardTitle>
      <CardDescription>Growth plan — renews May 11, 2026.</CardDescription>
    </div>
  </CardHeader>
</Card>;
```
