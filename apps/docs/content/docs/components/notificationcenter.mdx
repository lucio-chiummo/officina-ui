import { NotificationCenterDemo } from '@/components/demos/feedback-demo';

`NotificationCenter` is a bell button that shows an unread-count badge and opens
a floating panel listing your `notifications` (each needs an `id` and `title`,
plus optional `description`, `timestamp`, `unread`, and `onClick`). It wires up
mark-as-read, mark-all, clear, and see-all via the matching callbacks, sorts by
timestamp, and shows an `emptyState` when there's nothing left. Drive the list
from your own state and respond to the callbacks.

## Example

<Demo>
  <NotificationCenterDemo />
</Demo>

```tsx
import { NotificationCenter } from '@officina/ui';

<NotificationCenter
  notifications={[
    { id: '1', title: 'New comment', timestamp: new Date(), unread: true },
    { id: '2', title: 'Build passed', timestamp: new Date() },
  ]}
  onMarkRead={(id) => markRead(id)}
  onMarkAllRead={markAllRead}
  onSeeAll={() => navigate('/notifications')}
/>;
```
