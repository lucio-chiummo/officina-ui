---
title: Markdown
description: Render a Markdown string as sanitized, prose-styled HTML with GitHub-flavored syntax.
---

import { MarkdownDemo } from '@/components/demos/data-display-demo';

`Markdown` renders a Markdown string (passed as `children`) into prose-styled
HTML, supporting GitHub Flavored Markdown such as tables, task lists, and
strikethrough. Output is sanitized to strip unsafe HTML, so it is safe for
user-generated content like comments, descriptions, or AI responses.

## Example

<Demo>
  <MarkdownDemo />
</Demo>

```tsx
import { Markdown } from '@officina/ui';

function Notes() {
  return (
    <Markdown>{`# Release notes\n\n- **Fixed** a crash\n- Added \`Markdown\` support`}</Markdown>
  );
}
```
