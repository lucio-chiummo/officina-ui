import { ImageListItemDemo } from '@/components/demos/composed-demo';

`ImageListItem` is a single tile within an `ImageList` — a rounded, bordered
`figure` that frames its child image and renders an optional `caption` below it.
Put your `img` (or any media) as the child; pass `caption` for the figcaption.
It forwards any other `figure` props and merges `className`.

## Example

<Demo>
  <ImageListItemDemo />
</Demo>

```tsx
import { ImageListItem } from '@officina/ui';

<ImageListItem caption="Sunrise over the ridge">
  <img src="/photos/ridge.jpg" alt="Sunrise over the ridge" />
</ImageListItem>;
```
