`Gauge` is a lightweight, dependency-free SVG dial for a single value — handy
for KPI tiles and dashboards showing things like CPU or memory usage. Pass a
`value` (clamped between `min`/`max`, default `0`–`100`) and pick a `variant`:
`arc`, `semi`, or `linear`. Set `size`, a `label`, `thresholds` to switch colour
as the value crosses stops, or `formatValue` to add a unit. Unlike `GaugeChart`
it renders inline SVG, so no chart canvas or explicit height is needed.

## Example

<Demo>
  <GaugeDemo />
</Demo>

```tsx
import { Gauge } from '@officina/ui';

<Gauge value={72} label="CPU" size={140} />
<Gauge value={58} variant="semi" label="Memory" size={140} />;
```
