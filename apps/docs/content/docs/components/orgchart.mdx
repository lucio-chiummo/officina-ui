`OrgChart` draws a static, top-down tree of cards — pass a single recursive
`node` whose `children` branch out below it. Each node takes a `label` and
optional `description`. Use it for reporting structures, team layouts, or any
parent/child hierarchy you want to display at a glance. For an interactive,
expand/collapse hierarchy in a grid, see `TreeList` instead.

## Example

<Demo>
  <OrgChartDemo />
</Demo>

```tsx
import { OrgChart } from '@officina/ui';

function Reporting() {
  return (
    <OrgChart
      node={{
        id: 'ceo',
        label: 'Ada Lovelace',
        description: 'CEO',
        children: [
          { id: 'eng', label: 'Grace Hopper', description: 'VP Engineering' },
          { id: 'sales', label: 'Alan Turing', description: 'VP Sales' },
        ],
      }}
    />
  );
}
```
