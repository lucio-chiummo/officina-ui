import { TableCaptionDemo } from '@/components/demos/composed-demo';

`TableCaption` renders the table's `<caption>` — an accessible label describing
what the table contains. It appears as small, muted text below the table.
Include it as a direct child of `Table` so screen readers announce the table's
purpose.

## Example

<Demo>
  <TableCaptionDemo />
</Demo>

```tsx
import { Table, TableCaption } from '@officina/ui';

<Table>
  <TableCaption>Q3 invoices, sorted by issue date.</TableCaption>
  {/* …TableHeader, TableBody… */}
</Table>;
```
