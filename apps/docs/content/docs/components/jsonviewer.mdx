title: JSONViewer

import { JSONViewerDemo } from '@/components/demos/data-display-demo';

`JSONViewer` takes any `value` and renders it as an interactive tree with
syntax-highlighted keys and values; objects and arrays are collapsible nodes.
Set `defaultExpanded` to `false` to start collapsed. Use it for inspecting API
responses, config, or debug payloads in admin tools and dev panels.

## Example

<Demo>
  <JSONViewerDemo />
</Demo>

```tsx
import { JSONViewer } from '@officina/ui';

function ResponseInspector({ data }) {
  return <JSONViewer value={data} defaultExpanded />;
}
```
