---
title: TableRow
description: A <tr> row in a composed Table, with a hover highlight; wraps TableHead or TableCell cells.
---

import { TableRowDemo } from '@/components/demos/composed-demo';

`TableRow` renders a `<tr>` inside `Table` and applies a subtle hover
background. Use it for both the heading row (wrapping `TableHead` cells inside
`TableHeader`) and each data row (wrapping `TableCell` cells inside
`TableBody`). It forwards standard `<tr>` props, so attach `onClick` or `key`
directly.

## Example

<Demo>
  <TableRowDemo />
</Demo>

```tsx
import { TableBody, TableRow, TableCell } from '@officina/ui';

<TableBody>
  <TableRow onClick={() => openUser(user.id)}>
    <TableCell>{user.name}</TableCell>
    <TableCell>{user.email}</TableCell>
  </TableRow>
</TableBody>;
```
