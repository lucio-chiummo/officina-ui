---
title: ActivityFeed
description: Chronological feed of who did what and when, with actor avatars and timestamps.
---

import { ActivityFeedDemo } from '@/components/demos/data-display-demo';

`ActivityFeed` renders a list of activity entries, each pairing an `actor` (whose
initials become an avatar fallback) with an `action`, an optional `target`, and a
`timestamp`. Reach for it on dashboards, audit trails, or record detail pages to
show recent changes. Pass an `empty` node to control what shows when there is no
activity.

## Example

<Demo>
  <ActivityFeedDemo />
</Demo>

```tsx
import { ActivityFeed } from '@officina/ui';

function RecentActivity() {
  return (
    <ActivityFeed
      items={[
        {
          id: '1',
          actor: 'Ada Lovelace',
          action: 'closed the issue',
          target: 'OFF-218',
          timestamp: '2m ago',
        },
      ]}
      empty="No activity yet"
    />
  );
}
```
