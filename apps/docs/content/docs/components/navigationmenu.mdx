import { NavigationMenuDemo } from '@/components/demos/navigation-demo';

`NavigationMenu` is a horizontal top-level navigation bar built on Radix. Each
entry in `items` is either a simple link (`href`) or, when given `content`, a
trigger that opens a dropdown panel holding arbitrary JSX — featured links,
descriptions, or promos. Radix handles the keyboard navigation, focus, and
hover-intent timing. Use it for primary site or app navigation in a header.

## Example

<Demo>
  <NavigationMenuDemo />
</Demo>

```tsx
import { NavigationMenu } from '@officina/ui';

<NavigationMenu
  items={[
    { label: 'Home', href: '/' },
    { label: 'Products', content: <ProductsPanel /> },
    { label: 'Pricing', href: '/pricing' },
  ]}
/>;
```
