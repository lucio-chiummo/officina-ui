title: PageSection
description: Bordered content block with an optional title, description, and actions header.

import { PageSectionDemo } from '@/components/demos/composed-demo';

`PageSection` groups related content into a bordered, padded card with an
optional header. Provide a `title` and `description` for the heading, and an
`actions` slot for controls aligned to the right; the header is omitted entirely
when none are given. Stack several sections inside a
[`PageBody`](/docs/components/pagebody) to break a page into clear regions.

## Example

<Demo>
  <PageSectionDemo />
</Demo>

```tsx
import { PageSection, Button } from '@officina/ui';

<PageSection
  title="Team members"
  description="People with access to this workspace"
  actions={<Button size="sm">Invite</Button>}
>
  {/* table or list */}
</PageSection>;
```
