`Banner` announces system-wide messages across the top of a view.

## Example

<Demo>
  <BannerDemo />
</Demo>

```tsx
import { Banner } from '@officina/ui';

<Banner tone="warning" title="Scheduled maintenance" onDismiss={dismiss}>
  We'll be offline Sunday 02:00–03:00 UTC.
</Banner>;
```
