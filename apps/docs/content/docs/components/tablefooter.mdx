import { TableFooterDemo } from '@/components/demos/composed-demo';

`TableFooter` renders the `<tfoot>` of a hand-composed `Table`. It sits below
`TableBody` with a top border, muted background, and medium-weight text, making
it the place for totals, subtotals, or summary rows. Put a `TableRow` of
`TableCell`s inside it.

## Example

<Demo>
  <TableFooterDemo />
</Demo>

```tsx
import { Table, TableFooter, TableRow, TableCell } from '@officina/ui';

<Table>
  {/* …TableHeader, TableBody… */}
  <TableFooter>
    <TableRow>
      <TableCell>Total</TableCell>
      <TableCell className="text-right">$1,240.00</TableCell>
    </TableRow>
  </TableFooter>
</Table>;
```
