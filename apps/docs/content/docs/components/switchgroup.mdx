---
title: SwitchGroup
description: Stacked rows of labeled switches for settings panels, tracked as an array of on values.
---

import { SwitchGroupDemo } from '@/components/demos/inputs-demo';

`SwitchGroup` renders a list of `options` as labeled switch rows, ideal for
preference and settings panels. It is controlled — `value` is the array of values
currently switched on and `onChange` fires with the next array on every toggle.
Each option can carry a `description`, and by default the rows sit in a bordered,
divided panel (`bordered`, on by default).

## Example

<Demo>
  <SwitchGroupDemo />
</Demo>

```tsx
import { SwitchGroup } from '@officina/ui';
import { useState } from 'react';

function Example() {
  const [value, setValue] = useState<string[]>(['marketing']);
  return (
    <SwitchGroup
      value={value}
      onChange={setValue}
      options={[
        { value: 'marketing', label: 'Marketing emails', description: 'Product news and offers.' },
        { value: 'security', label: 'Security alerts' },
      ]}
    />
  );
}
```
