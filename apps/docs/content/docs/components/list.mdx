`List` is the styled `<ul>` that wraps a stack of `ListItem` rows. It owns the
shared spacing and surface — pass `variant="outlined"` or `"soft"` for a bordered
or filled container, and `dense` to tighten the row gap. Fill it with `ListItem`,
`ListSubheader`, and `ListDivider` children to build navigation menus, settings
panels, and entity lists.

## Example

<Demo>
  <ListDemo />
</Demo>

```tsx
import { List, ListItem, ListItemButton } from '@officina/ui';

<List variant="outlined">
  <ListItem>
    <ListItemButton selected>Dashboard</ListItemButton>
  </ListItem>
  <ListItem>
    <ListItemButton>Settings</ListItemButton>
  </ListItem>
</List>;
```
