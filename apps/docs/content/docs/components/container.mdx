title: Container

import { ContainerDemo } from '@/components/demos/misc-demo';

`Container` constrains content to a comfortable width and centers it
horizontally, with responsive side padding built in. Pick a max-width with the
`size` prop (`sm`, `md`, `lg`, `xl` default, `2xl`, or `full` for none). Wrap
page bodies and section content in it so layouts stay aligned to the same
measure across breakpoints.

## Example

<Demo>
  <ContainerDemo />
</Demo>

```tsx
import { Container } from '@officina/ui';

<Container size="lg">
  <h1>Dashboard</h1>
  {/* page content */}
</Container>;
```
