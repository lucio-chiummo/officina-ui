---
title: ImageCropper
description: Interactive image cropper that returns the selected region as an image Blob.
---

import { ImageCropperDemo } from '@/components/demos/composed-demo';

`ImageCropper` lets users drag a selection over an image and confirm a crop —
ideal for avatar and cover-photo flows. Pass the `src` to crop and an optional
`aspect` ratio (width / height) to lock the selection shape; leave it off for a
free-form crop. On confirm, `onCrop` is called with the cropped region as a
`Blob` you can upload or preview.

## Example

<Demo>
  <ImageCropperDemo />
</Demo>

```tsx
import { ImageCropper } from '@officina/ui';

<ImageCropper
  src={objectUrl}
  aspect={1}
  cropLabel="Save crop"
  onCrop={(blob) => uploadAvatar(blob)}
/>;
```
