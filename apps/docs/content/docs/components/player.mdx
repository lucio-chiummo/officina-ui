---
title: Player
description: Responsive video/audio player wrapping react-player, with poster and native controls.
---

import { PlayerDemo } from '@/components/demos/misc-demo';

`Player` embeds video or audio from a URL in a 16:9 frame, wrapping `react-player`
so it handles file URLs as well as sources like YouTube and Vimeo. Pass the
media `src`, an optional `poster` image shown before playback, and toggle
`controls` (on by default). It is loaded lazily, so it only ships its player
code when rendered.

## Example

<Demo>
  <PlayerDemo />
</Demo>

```tsx
import { Player } from '@officina/ui';

<Player src="https://example.com/demo.mp4" poster="/poster.jpg" controls />;
```
