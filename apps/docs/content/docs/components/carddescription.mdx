`CardDescription` is the small, muted supporting line that sits under a
`CardTitle` — a styled `<p>` for a subtitle, status, or one-line summary. Place
it inside a `CardHeader` next to the title.

## Example

<Demo>
  <CardDescriptionDemo />
</Demo>

```tsx
import { Card, CardHeader, CardTitle, CardDescription } from '@officina/ui';

<Card>
  <CardHeader>
    <div>
      <CardTitle>Acme Operations</CardTitle>
      <CardDescription>Growth plan — renews May 11, 2026.</CardDescription>
    </div>
  </CardHeader>
</Card>;
```
