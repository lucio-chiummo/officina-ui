title: Box

import { BoxDemo } from '@/components/demos/misc-demo';

`Box` is the lowest-level layout primitive — a `div` with `min-w-0` already
applied (so flex and grid children can shrink and truncate correctly) and no
other styling. Use it as a neutral styling target you hang `className` on, or
render it as any element with the `as` prop (`section`, `aside`, `nav`, …).
Reach for it when you want a plain container without the spacing opinions of
`Stack` or `Container`.

## Example

<Demo>
  <BoxDemo />
</Demo>

```tsx
import { Box } from '@officina/ui';

<Box as="section" className="rounded-lg border p-4">
  Content
</Box>;
```
