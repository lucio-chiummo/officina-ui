---
title: Typography
description: Polymorphic text component with a full variant, tone, and weight scale.
---

import { TypographyScale } from '@/components/demos/typography-demo';

`Typography` renders text on the design scale, with `as` for the element.

## Example

<Demo>
  <TypographyScale />
</Demo>

```tsx
import { Typography } from '@officina/ui';

<Typography variant="h2">Heading</Typography>
<Typography variant="body" tone="muted">Secondary copy</Typography>
```
