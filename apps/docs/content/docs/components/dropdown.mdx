import { DropdownDemo } from '@/components/demos/navigation-demo';

`Dropdown` is the low-level menu primitive the other menus build on. Pass any
`trigger` element and a list of `sections`, each an optional `heading` plus
items with `label`, `icon`, `shortcut`, `danger`, and `onClick`/`href`. The
panel anchors to the trigger and aligns to `start` or `end`. Built on Headless
UI, so keyboard navigation, focus management, and outside-click dismissal come
for free.

## Example

<Demo>
  <DropdownDemo />
</Demo>

```tsx
import { Dropdown, Button } from '@officina/ui';

<Dropdown
  trigger={<Button variant="secondary">Options</Button>}
  align="end"
  sections={[
    {
      items: [
        { label: 'Rename', onClick: rename },
        { label: 'Delete', danger: true, onClick: remove },
      ],
    },
  ]}
/>;
```
