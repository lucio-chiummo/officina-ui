`FilterGroupEditor` is the advanced query builder for complex filters. It lets
users nest conditions into AND/OR groups (up to `maxDepth`), combining multiple
`FilterBuilder` rows into a tree. It reads and updates the shared state from
`FiltersProvider`, so it takes only the optional `fields` list and a depth
limit. The panel is lazy-loaded; `FilterBar` reveals it automatically once a
group exists.

## Example

<Demo>
  <FilterGroupEditorDemo />
</Demo>

```tsx
import { FiltersProvider, FilterGroupEditor, useFilters } from '@officina/ui';

<FiltersProvider value={filters}>
  <FilterGroupEditor
    fields={[
      { id: 'status', label: 'Status', type: 'enum' },
      { id: 'amount', label: 'Amount', type: 'number' },
    ]}
    maxDepth={2}
  />
</FiltersProvider>;
```
