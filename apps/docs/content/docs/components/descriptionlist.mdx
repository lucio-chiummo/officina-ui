---
title: DescriptionList
description: Data-driven term/value rows rendered as a semantic dl, inline or stacked.
---

import { DescriptionListDemo } from '@/components/demos/data-display-demo';

`DescriptionList` renders an array of `term`/`detail` pairs as a semantic
`<dl>` with a top and bottom rule on each row. Pass `layout="inline"` (the
default) to set the term in a fixed-width column beside its value, or
`layout="stacked"` to put the term above the value. Reach for it on detail and
summary screens where the data is a known set of fields.

## Example

<Demo>
  <DescriptionListDemo />
</Demo>

```tsx
import { DescriptionList } from '@officina/ui';

<DescriptionList
  items={[
    { term: 'Plan', detail: 'Pro' },
    { term: 'Seats', detail: '24 / 50' },
    { term: 'Renewal', detail: 'May 11, 2026' },
  ]}
/>;
```
