title: RangeSlider

import { RangeSliderDemo } from '@/components/demos/inputs-demo';

`RangeSlider` is a two-thumb slider for choosing a span — a min and a max — such
as a price or date range filter. It is controlled: `value` is a `[number,
number]` tuple and `onValueChange` returns the updated tuple. Tune the bounds
with `min`, `max`, and `step`, keep the thumbs apart with
`minStepsBetweenThumbs`, and format the visible numbers with `formatValue`.

## Example

<Demo>
  <RangeSliderDemo />
</Demo>

```tsx
import { RangeSlider } from '@officina/ui';
import { useState } from 'react';

function PriceFilter() {
  const [range, setRange] = useState<[number, number]>([20, 80]);
  return (
    <RangeSlider
      label="Price range"
      value={range}
      onValueChange={setRange}
      min={0}
      max={100}
      minStepsBetweenThumbs={1}
      formatValue={(n) => `$${n}`}
    />
  );
}
```
