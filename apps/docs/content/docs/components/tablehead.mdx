`TableHead` is a header cell (`<th>`) for a hand-composed `Table`. It renders
the column label in small, uppercase, muted text, left-aligned by default. Use
one per column inside the `TableRow` of your `TableHeader`. Do not confuse it
with `TableHeader`, which is the `<thead>` section that wraps these cells.

## Example

<Demo>
  <TableHeadDemo />
</Demo>

```tsx
import { TableHeader, TableRow, TableHead } from '@officina/ui';

<TableHeader>
  <TableRow>
    <TableHead>Name</TableHead>
    <TableHead className="text-right">Amount</TableHead>
  </TableRow>
</TableHeader>;
```
