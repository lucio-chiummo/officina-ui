`Paper` is a styled surface for grouping content onto a card-like background. The
`variant` prop sets its edge treatment (`outlined` default, `elevated`, or
`subtle`), and `elevation` (`0`–`3`) sets the shadow depth. Use it as the base
for cards, popovers, and panels when you want a consistent surface without
wiring up borders and shadows by hand.

## Example

<Demo>
  <PaperDemo />
</Demo>

```tsx
import { Paper } from '@officina/ui';

<Paper variant="elevated" elevation={2} className="p-4">
  Card content
</Paper>;
```
