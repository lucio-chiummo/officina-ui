title: CardContent

import { CardContentDemo } from '@/components/demos/composed-demo';

`CardContent` is the body region of a `Card` — an unstyled wrapper that marks
the main content between the header and footer. Put your primary content (text,
fields, charts) here so the card's sections read in a predictable order.

## Example

<Demo>
  <CardContentDemo />
</Demo>

```tsx
import { Card, CardHeader, CardTitle, CardContent } from '@officina/ui';

<Card>
  <CardHeader>
    <CardTitle>Acme Operations</CardTitle>
  </CardHeader>
  <CardContent>
    <p>Compose header, content, and footer to build a card surface.</p>
  </CardContent>
</Card>;
```
