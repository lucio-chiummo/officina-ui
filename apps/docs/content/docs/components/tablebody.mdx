title: TableBody

import { TableBodyDemo } from '@/components/demos/composed-demo';

`TableBody` renders the `<tbody>` of a hand-composed `Table` and contains the
data rows. It draws a thin divider between each `TableRow` so rows read clearly.
Place it after `TableHeader` and map your records to `TableRow`/`TableCell`
inside it.

## Example

<Demo>
  <TableBodyDemo />
</Demo>

```tsx
import { Table, TableBody, TableRow, TableCell } from '@officina/ui';

<Table>
  {/* …TableHeader… */}
  <TableBody>
    {users.map((user) => (
      <TableRow key={user.id}>
        <TableCell>{user.name}</TableCell>
        <TableCell>{user.role}</TableCell>
      </TableRow>
    ))}
  </TableBody>
</Table>;
```
