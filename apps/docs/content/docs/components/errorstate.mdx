---
title: ErrorState
description: Danger-toned placeholder for failed loads, with a message and retry action.
---

import { ErrorStateDemo } from '@/components/demos/feedback-demo';

`ErrorState` fills a region that failed to load with a danger-toned box: a
`title` (default `'Something went wrong'`), an optional `message`, and an
optional `retry` action. It carries `role="alert"` so assistive tech announces
it. Use it as the error branch when a panel, list, or query fails.

## Example

<Demo>
  <ErrorStateDemo />
</Demo>

```tsx
import { ErrorState, Button } from '@officina/ui';

<ErrorState
  title="Couldn't load orders"
  message="Check your connection and try again."
  retry={<Button onClick={refetch}>Retry</Button>}
/>;
```
