import { TableCellDemo } from '@/components/demos/composed-demo';

`TableCell` is a data cell (`<td>`) for a hand-composed `Table`. It applies the
standard cell padding and vertical alignment so columns line up. Use one per
column inside each `TableRow` in your `TableBody`; pass `colSpan`, `className`,
or any `<td>` attribute as needed.

## Example

<Demo>
  <TableCellDemo />
</Demo>

```tsx
import { TableBody, TableRow, TableCell } from '@officina/ui';

<TableBody>
  <TableRow>
    <TableCell>Ada Lovelace</TableCell>
    <TableCell className="text-right">$250.00</TableCell>
  </TableRow>
</TableBody>;
```
