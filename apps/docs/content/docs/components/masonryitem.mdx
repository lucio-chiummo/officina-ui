---
title: MasonryItem
description: Wrapper for a single Masonry child that prevents it from breaking across columns.
---

import { MasonryItemDemo } from '@/components/demos/composed-demo';

`MasonryItem` wraps each child inside a [`Masonry`](/docs/components/masonry)
container. Because `Masonry` uses CSS columns, an unwrapped element can be split
across a column break; `MasonryItem` applies `break-inside: avoid` and bottom
spacing so each item stays whole and evenly separated. Always render your
masonry children inside it.

## Example

<Demo>
  <MasonryItemDemo />
</Demo>

```tsx
import { Masonry, MasonryItem } from '@officina/ui';

<Masonry columns={3} gap={12}>
  {items.map((item) => (
    <MasonryItem key={item.id}>
      <Card>{item.title}</Card>
    </MasonryItem>
  ))}
</Masonry>;
```
