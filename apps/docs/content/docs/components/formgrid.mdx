---
title: FormGrid
description: Responsive grid that arranges form fields into 1–4 columns that collapse on small screens.
---

import { FormGridDemo } from '@/components/demos/forms-demo';

`FormGrid` lays out fields in a responsive CSS grid. Pick `columns` between 1 and
4 (defaults to 2); the grid drops to a single column on narrow viewports and
expands at the configured breakpoints, so fields like first/last name sit side by
side on desktop and stack on mobile.

## Example

<Demo>
  <FormGridDemo />
</Demo>

```tsx
import { FormGrid, Field, Input } from '@officina/ui';

<FormGrid columns={3}>
  <Field label="City">
    <Input />
  </Field>
  <Field label="State">
    <Input />
  </Field>
  <Field label="ZIP">
    <Input />
  </Field>
</FormGrid>;
```
