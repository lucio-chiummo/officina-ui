title: TableHeader

import { TableHeaderDemo } from '@/components/demos/composed-demo';

`TableHeader` renders the `<thead>` of a hand-composed `Table`. It holds the
single `TableRow` of column headings (`TableHead` cells) and adds a bottom
border to separate the head from the body. Use it as the first child of
`Table`, before `TableBody`.

## Example

<Demo>
  <TableHeaderDemo />
</Demo>

```tsx
import { Table, TableHeader, TableRow, TableHead } from '@officina/ui';

<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Name</TableHead>
      <TableHead>Role</TableHead>
    </TableRow>
  </TableHeader>
  {/* …TableBody… */}
</Table>;
```
