import { PrimitiveLinkDemo } from '@/components/demos/misc-demo';

`PrimitiveLink` is a styled `<a>` with consistent focus rings and accent
coloring. Set `muted` for a quieter foreground tone, or `external` to open in a
new tab — which also adds safe `rel` attributes and an external-link icon
automatically. It truncates long labels and accepts any standard anchor prop, so
it works as a drop-in for inline links and CTAs.

## Example

<Demo>
  <PrimitiveLinkDemo />
</Demo>

```tsx
import { PrimitiveLink } from '@officina/ui';

<PrimitiveLink href="/pricing">View pricing</PrimitiveLink>
<PrimitiveLink href="https://example.com" external muted>
  Docs
</PrimitiveLink>
```
