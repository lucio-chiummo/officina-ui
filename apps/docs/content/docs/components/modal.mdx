`Modal` is a portaled, centered dialog for focused tasks that should block the
rest of the page — confirmations, forms, detail views. It renders a dimmed
backdrop, sets `role="dialog"` with `aria-modal`, and dismisses on `Escape`,
backdrop click, or the close button. It is controlled: drive `open` from your
own state and clear it in `onClose`.

## Example

<Demo>
  <ModalDemo />
</Demo>

```tsx
import { Modal } from '@officina/ui';
import { useState } from 'react';

function EditDialog() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <button onClick={() => setOpen(true)}>Edit</button>
      <Modal
        open={open}
        onClose={() => setOpen(false)}
        title="Edit profile"
        description="Update your account details."
      >
        {/* form fields */}
      </Modal>
    </>
  );
}
```
