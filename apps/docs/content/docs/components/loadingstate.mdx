import { LoadingStateDemo } from '@/components/demos/feedback-demo';

`LoadingState` is a minimal inline indicator — a spinner next to a `label` —
for showing that a panel or section is loading. It announces itself with
`role="status"` and `aria-live="polite"`. Use it where you want a quiet,
in-flow loading cue rather than an overlay; the `label` defaults to `'Loading…'`.

## Example

<Demo>
  <LoadingStateDemo />
</Demo>

```tsx
import { LoadingState } from '@officina/ui';

{
  isLoading ? <LoadingState label="Fetching results…" /> : <Results />;
}
```
