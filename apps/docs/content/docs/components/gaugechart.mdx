title: GaugeChart

import { GaugeChartDemo } from '@/components/demos/charts-demo';

`GaugeChart` renders a single value as a dial arc against a range — use it for
one headline number measured against a target, such as quota attainment or
utilization. Pass the `value`, optionally set `min`/`max` (default `0`–`100`)
and a `label`. It renders client-side via ECharts, so give it a `height`
(defaults to `280`). For a lighter SVG dial or progress bar, see `Gauge`.

## Example

<Demo>
  <GaugeChartDemo />
</Demo>

```tsx
import { GaugeChart } from '@officina/ui';

<GaugeChart value={72} label="Quota" max={100} height={280} />;
```
