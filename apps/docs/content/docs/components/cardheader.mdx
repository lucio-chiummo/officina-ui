`CardHeader` is the top section of a `Card`. It lays its children out in a row
with space between them, so a `CardTitle`/`CardDescription` block sits on the
left while an action button or menu sits on the right, with padding below to
separate it from the content. Place it as the first child of a `Card`.

## Example

<Demo>
  <CardHeaderDemo />
</Demo>

```tsx
import { Card, CardHeader, CardTitle, CardDescription, Button } from '@officina/ui';

<Card>
  <CardHeader>
    <div>
      <CardTitle>Acme Operations</CardTitle>
      <CardDescription>Growth plan — renews May 11, 2026.</CardDescription>
    </div>
    <Button size="sm" variant="soft">
      Manage
    </Button>
  </CardHeader>
</Card>;
```
