---
title: ToggleButton
description: A button with a controlled on/off pressed state, exposed via aria-pressed.
---

import { ToggleButtonDemo } from '@/components/demos/buttons-demo';

`ToggleButton` is a `Button` with a binary pressed state — think bold, mute, or
pin. It is fully controlled: pass `pressed` and handle `onPressedChange`, which
fires with the next value. The pressed state is reflected with `aria-pressed`
and an accent style, and it forwards the rest of the `Button` props.

## Example

<Demo>
  <ToggleButtonDemo />
</Demo>

```tsx
import { ToggleButton } from '@officina/ui';
import { useState } from 'react';

function BoldToggle() {
  const [bold, setBold] = useState(false);
  return (
    <ToggleButton pressed={bold} onPressedChange={setBold}>
      Bold
    </ToggleButton>
  );
}
```
