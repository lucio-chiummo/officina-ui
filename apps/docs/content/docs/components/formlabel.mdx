---
title: FormLabel
description: Label for a FormControl that auto-links to its control and shows a required indicator.
---

import { FormLabelDemo } from '@/components/demos/composed-demo';

`FormLabel` is the `<label>` for a `FormControl`. Inside one it inherits the
control's id (so `htmlFor` points at the right input) and its `required` flag,
rendering a `*` indicator when the field is required. You can also pass `htmlFor`
or `required` explicitly to use it standalone.

## Example

<Demo>
  <FormLabelDemo />
</Demo>

```tsx
import { FormControl, FormLabel, Input } from '@officina/ui';

<FormControl required>
  <FormLabel>Workspace name</FormLabel>
  <Input />
</FormControl>;
```
