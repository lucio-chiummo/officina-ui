`PanelHandle` is the draggable separator that sits between two
[`Panel`](/docs/components/panel) regions in a
[`ResizablePanels`](/docs/components/resizablepanels) group. Dragging it
reallocates space between the neighboring panels. It comes pre-styled with a
grip indicator and hover, focus, and active states, and orients itself
automatically to the group's direction. Place one between every pair of panels.

## Example

<Demo>
  <ResizablePanelsDemo />
</Demo>

```tsx
import { ResizablePanels, Panel, PanelHandle } from '@officina/ui';

<ResizablePanels direction="horizontal">
  <Panel>Left</Panel>
  <PanelHandle />
  <Panel>Right</Panel>
</ResizablePanels>;
```
