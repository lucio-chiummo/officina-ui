import { ListItemDemo } from '@/components/demos/composed-demo';

`ListItem` is one row (`<li>`) inside a `List`. It is a thin, presentational
wrapper that handles rounding and min-width — put plain content inside for a
static row, or a `ListItemButton` for an interactive one. Compose it with
`ListItemDecorator` and `ListItemContent` to build richer rows.

## Example

<Demo>
  <ListItemDemo />
</Demo>

```tsx
import { List, ListItem, ListItemButton } from '@officina/ui';

<List>
  <ListItem>
    <ListItemButton>Dashboard</ListItemButton>
  </ListItem>
  <ListItem>
    <ListItemButton>Settings</ListItemButton>
  </ListItem>
</List>;
```
