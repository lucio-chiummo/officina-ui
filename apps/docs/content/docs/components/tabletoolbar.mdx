`TableToolbar` is the control strip you place above a table. It lays out a
built-in search box on the left (via the `search` config), `filters` next to it,
and pins `bulkActions`, a `columnsToggle`, and any `rightSlot` content to the
right edge — collapsing to a stacked layout on small screens. Use it to give
`DataTable`/`DataGrid` or a hand-built `Table` a consistent header of controls.

## Example

<Demo>
  <TableToolbarDemo />
</Demo>

```tsx
import { TableToolbar, Button } from '@officina/ui';
import { useState } from 'react';

function ListToolbar() {
  const [query, setQuery] = useState('');
  return (
    <TableToolbar
      search={{ value: query, onChange: setQuery, placeholder: 'Search users…' }}
      rightSlot={<Button size="sm">New user</Button>}
    />
  );
}
```
