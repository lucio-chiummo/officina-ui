title: ImageLightbox

import { ImageLightboxDemo } from '@/components/demos/misc-demo';

`ImageLightbox` opens a full-screen, navigable overlay for viewing images at
size — wrap it around a thumbnail grid so clicking a photo opens the gallery.
It is controlled: drive `open` from your state, supply the `slides` array, set
`startIndex` for which one opens first, and reset `open` in `onClose`.

## Example

<Demo>
  <ImageLightboxDemo />
</Demo>

```tsx
import { ImageLightbox } from '@officina/ui';
import { useState } from 'react';

function Gallery() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <button onClick={() => setOpen(true)}>View photos</button>
      <ImageLightbox
        open={open}
        onClose={() => setOpen(false)}
        startIndex={0}
        slides={[{ src: '/photos/1.jpg' }, { src: '/photos/2.jpg' }]}
      />
    </>
  );
}
```
