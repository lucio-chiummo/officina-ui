---
title: ListDivider
description: A horizontal separator rule between groups of list rows.
---

import { ListDividerDemo } from '@/components/demos/composed-demo';

`ListDivider` draws a thin horizontal rule as a `<li role="separator">` between
rows of a `List`. Drop it between groups of items to visually separate sections
without a heading. It is purely decorative and takes no content.

## Example

<Demo>
  <ListDividerDemo />
</Demo>

```tsx
import { List, ListItem, ListItemButton, ListDivider } from '@officina/ui';

<List>
  <ListItem>
    <ListItemButton>Dashboard</ListItemButton>
  </ListItem>
  <ListDivider />
  <ListItem>
    <ListItemButton>Settings</ListItemButton>
  </ListItem>
</List>;
```
