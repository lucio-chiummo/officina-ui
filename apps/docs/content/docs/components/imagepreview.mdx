import { ImagePreviewDemo } from '@/components/demos/misc-demo';

`ImagePreview` is a square thumbnail tile for files being uploaded — it shows the
image plus optional remove and retry controls, and overlays a progress or error
state. Pass the `src` and an optional `name`; drive the overlay with `status`
(`'idle' | 'uploading' | 'error'`) and `progress`. It is transport-free, so you
own the object URL and wire `onRemove`/`onRetry` yourself.

## Example

<Demo>
  <ImagePreviewDemo />
</Demo>

```tsx
import { ImagePreview } from '@officina/ui';

<ImagePreview
  src={objectUrl}
  name="avatar.png"
  status="uploading"
  progress={64}
  onRemove={() => removeFile(id)}
  onRetry={() => retryUpload(id)}
/>;
```
