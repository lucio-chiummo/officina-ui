import { ListItemContentDemo } from '@/components/demos/composed-demo';

`ListItemContent` is the text body of a row. Pass a `title` and optional
`description` and it renders them as a primary line plus a muted secondary line,
each truncated to a single line so long values never break the layout. It flexes
to fill the remaining space, so place it after a `ListItemDecorator` inside a
`ListItemButton`. You can also pass `children` for fully custom content.

## Example

<Demo>
  <ListItemContentDemo />
</Demo>

```tsx
import { List, ListItem, ListItemButton, ListItemContent } from '@officina/ui';

<List>
  <ListItem>
    <ListItemButton>
      <ListItemContent title="Dashboard" description="Overview and KPIs" />
    </ListItemButton>
  </ListItem>
</List>;
```
