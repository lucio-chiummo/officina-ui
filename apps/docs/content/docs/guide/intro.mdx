Officina is a production-grade React admin/dashboard template backed by
`@officina/ui` — an MIT-licensed library of 240+ React primitives. Everything
needed to ship is included; there is no separate component kit to license or
download. It targets:

- Solo SaaS founders shipping their MVP admin panel.
- Agencies that resell admin work to clients.
- In-house teams that want a sane internal tool starter.

## What it includes

- React 19 (SPA, no SSR)
- TypeScript 5.6 strict (`noUncheckedIndexedAccess`, `exactOptionalPropertyTypes`)
- Vite 6 + Tailwind v4
- TanStack Router (file-based, type-safe) + Query (server state) + Table (DataTable)
- React Hook Form + Zod (forms + schemas at the network boundary)
- Zustand (client-only state slices)
- MSW (in-browser + Node mocks for the entire API surface)
- Sonner toaster, cmdk command palette, Headless UI dialogs
- Vitest + Playwright + jest-axe + Storybook 8

## UI primitives

All visual primitives ship in `@officina/ui` (MIT) — buttons, inputs, tables,
charts, overlays, and more. Import them directly; no external kit or license is
required. See the [Component Documentation](/docs/components) for live demos and
API tables.
