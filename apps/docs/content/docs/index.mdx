Officina UI is a production-grade React component library: **250+ primitives**
covering forms, data display, overlays, charts, scheduling, and AI surfaces —
all token-driven, accessible, and typed.

## Why Officina

- **Complete.** Everything from `Button` to `Scheduler`, `TreeList`, `Gauge`,
  and `CandlestickChart` — parity with commercial suites, MIT-licensed.
- **Token-driven theming.** Every component reads CSS custom properties. Swap a
  handful of variables to rebrand the whole system.
- **Typed and documented.** Prop tables on this site are generated directly from
  the source types — they cannot drift from the code.
- **Accessible by default.** ARIA roles, keyboard navigation, and focus
  management are built in, not bolted on.
