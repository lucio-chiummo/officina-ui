Side-nav layout under `/settings/*`. Sub-pages:

- **Profile** — name / bio / timezone (RHF + Zod).
- **Security** — change password, 2FA toggle, recovery codes, active
  sessions list with revoke.
- **Appearance** — theme mode (light / dark / system), preset brand-color
  swatches + custom color picker, density toggle. Persists via the theme
  store.
- **Notifications** — event × channel matrix (5 events × 3 channels:
  email / in-app / mobile).
- **API keys** — list / create / revoke. Created key reveals once.
- **Webhooks** — list / create with event-tag chips / test ping with recent
  deliveries inline / delete.
- **Locale** — language / date format / first-day-of-week.
- **Roles & permissions** — RBAC matrix viewer (Phase 6).
- **Danger zone** — delete account via typed-confirm with the user's email.
