# Security Policy

## Supported versions

Officina UI is pre-1.0 and tracks a single line of development on `main`.
Security fixes are released against the latest published version of
`@officina/ui`.

## Reporting a vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Instead, report it privately via
[GitHub Security Advisories](https://github.com/lucio-chiummo/officina-ui/security/advisories/new)
or by emailing l.chiummo.dev@gmail.com. Include:

- A description of the vulnerability and its impact
- Steps to reproduce or a proof of concept
- Any suggested remediation, if known

We'll acknowledge your report and aim to provide a fix or mitigation timeline
as soon as possible.
